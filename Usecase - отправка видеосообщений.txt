Словарь:
	1. Пользователь - студент или преподаватель
	2. Отправка видеосообщения:
		a. Переслать сообщение с приложенным к нему видео
		b. Записать новое видео с последующей его отправкой
		c. Выбрать видео с памяти устройства для последующей отправки
	3. Система - приложение-мессенджер
	4. Получатель - пользователь, имеющий право просматривать полученное видео
	5. Устройство - девайс, с которого запускают систему
UseCase - Отправка видеосообщений

Основное действующие лица - пользователь, система

Цель - отправить видеосообщение

Триггер - пользователь решает отправить видеосообщение

Результат - получатель сообщения может просмотреть видеосообщение

Действия системы и пользователя пошагово для отправки видеосообщениия пункта 2.а словаря:
	1. Пользователь - открывает приложение
	2. Пользователь - открывает чат, в котором находится сообщение с вложенным видео
	3. Система - получает историю сообщений для этого чата
	4. Пользователь - выбирает сообщение
	5. Система - запоминает выбор пользователя
	6. Пользователь - выбирает опцию - "переслать сообщение"
	7. Система - возвращает список чатов пользователя
	8. Пользователь - выбирает получателя (нажимает на чат с пользователем - получателем)
	9. Система - просит ввести доп. данные (подпись к пересылаемому сообщению)
	10. Пользователь - Нажимает на кнопку "Отправить"
	?11. Система предоставляет доступ к хранилищу для именно этого видео пользователю-получателю (чтобы лишний раз не загружать его заново) - зависит от usecase  - Хранение данных
	
Действия системы и пользователя пошагово для отправки видеосообщениия пункта 2.b словаря:
	1. Пользователь - открывает приложение
	2. Пользователь - открывает чат с получателем
	3. Система - получает историю сообщений для этого чата
	4. Пользователь - выбирает опцию - "записать видео" (нажимает на кнопку - "записать видео")
	5. Система - получает доступ к камере и микрофону устройства
	6. Пользователь - Пока не отжата кнопка, записывает видео
	7. Пользователь - отжимает кнопку
	8. Система - завершает запись видео
	9. Система - просит ввести доп. данные (подпись к отправляемому сообщению)
	10. Пользователь - Нажимает кнопку "Отправить"
	11. Система - в чате показывает, что сообщение отправляется (пока видео не загружено в хранилище, сообщение не может отображаться у получателя)
	12. Система - Загружает видео в хранилище
	13. Система - по завршении загрузки видео, предоставляет доступ к хранилищу для именно этого видео получателю

Действия системы и пользователя пошагово для отправки видеосообщениия пункта 2.c словаря:
	1. Пользователь - открывает приложение
	2. Пользователь - открывает чат с получателем
	3. Система - получает историю сообщений для этого чата
	4. Пользователь - выбирает опцию - "прикрепить файл"
	?5. Система - показывает список возможных видов вложений(документ, фото, видео, просто открытие хранилища устройства и тд.) - зависит от usecase - Вложение в сообщение
	6. Система - получает доступ к хранилищу устройства
	7. Ситема - открывает "проводник" устройства
	8. Пользователь - выбирает нужное видео
	9. Система - просит ввести доп. данные (подпись к отправляемому сообщению)
	10. Пользователь - Нажимает кнопку "Отправить"
	11. Система - в чате показывает, что сообщение отправляется (пока видео не загружено в хранилище, сообщение не может отображаться у получателя)
	12. Система - Загружает видео в хранилище
	13. Система - по завршении загрузки видео, предоставляет доступ к хранилищу для именно этого видео пользователю-получателю
